# 项目规划（Plan）

## 一、项目定位

企业内部培训与技能评估平台（LMS），以图文 / 文档类课程为主，不依赖视频播放功能，面向以下场景：

- 管理员配置角色、课程与考试规则。
- 讲师创建课程内容、题库和考试。
- 学员在线浏览课程文档、完成学习与考试。

当前状态：基础 RBAC（用户 / 角色 / 权限 + 关联表）已搭建完成。

## 二、核心功能模块

1. 认证与权限
   - JWT 登录 / 注册接口。
   - 基于角色的路由控制（admin / instructor / student）。

2. 用户与角色管理
   - 用户列表、详情查询接口。
   - 管理员给用户分配 / 取消角色。

3. 课程与内容模块（图文为主）
   - 课程 CRUD：标题、简介、难度、标签、状态（草稿/发布）。
   - 课程下的章节 / 小节（文档内容、外部链接）。
   - 课程列表 / 搜索 / 详情接口。

4. 报名与学习进度
   - 学员报名 / 取消报名课程。
   - 查询“我的课程”、课程学习进度（完成章节数 / 总章节数）。

5. 题库与考试
   - 题库：选择题 / 判断题 / 简答题的 CRUD。
   - 考试：从题库组卷，配置时间与及格线。
   - 提交答卷、自动判分（客观题），记录成绩与错题。

6. 报表与统计（后期）
   - 按课程统计报名人数、完成率、平均成绩。
   - 按学员统计完成课程数、考试成绩概况。

## 三、里程碑划分

- M1：JWT 认证 + 基于角色的访问控制 + 用户基础接口。
- M2：课程与章节模型设计 + 课程 CRUD + 列表 / 详情接口。
- M3：课程报名与学习进度接口（含“我的课程”）。
- M4：题库与考试模块（组卷、交卷、判分、成绩查询）。
- M5：报表接口与性能/权限细化优化。
